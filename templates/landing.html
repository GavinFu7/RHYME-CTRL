FROM MP3 TO SYNCED,
RHYMED LYRIC VIDEO.

Start with audio or text. Whisper transcribes and aligns words to the beat. The rhyme engine finds families. Performance mode shows everything in sync. Finetune emphasis, then render a video.

PROCESSING PIPELINE

1
Start from MP3
Drop in an MP3 and let Whisper generate baseline lyrics. Clean up the transcript before alignment, or skip this if you already have lyrics.
TRANSCRIBE FROM AUDIO →
2
Edit & explore rhymes
Paste the lyrics (ideally one bar per line). Explore rhyme families at the word, sub-word and phoneme level. Fix line breaks until it feels right.
OPEN TEXT MODE
3
Align audio & watch performance
Upload audio + lyrics. The app aligns bars and words via Whisper. Performance mode highlights each word in sync; use the offset slider if things feel early or late.
STATIC AUTO-ALIGN LIVE PERFORMANCE
4
Finetune emphasis for video
On top of the AI baseline, manually mark emphasis (0 / 1 / 2) on internal rhymes, punchlines and key words. Export JSON with timing + emphasis for video tools.
OPEN FINETUNE EDITOR
5
Render a lyric video
Use the CLI renderer to turn your MP3 + finetune JSON into a simple lyric video. This is your one-command "run the whole pipeline" step.
# 1. Export JSON from Finetune mode
# 2. Then run in your venv:

python3 render_video.py \
  --audio /path/to/track.mp3 \
  --json alignment.json \
  --out track_lyrics.mp4
